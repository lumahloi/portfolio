# Tweet Deleter
Script que acessa sua conta do Twitter/X e inicia o Selenium para excluir seus tweets. Atualmente, as credenciais de login precisam ser atualizadas diretamente no código, mas isso será alterado no futuro.

## Pré-requisitos
- **Python**: verifique a instalação no [site oficial](https://www.python.org/downloads/)

## Instalação
- **Dependências de download**
```bash
pip install -r requirements.txt
```

## Uso
Altere as seguintes variáveis ​​no código:
1. **usuário**: seu nome de usuário;
2. **password**: sua senha;
3. **URL_DA_PAGINA**: a URL do seu perfil.

## Executar
- **Executar o projeto localmente**
```bash
py file.py
```

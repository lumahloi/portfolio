# BookBuddy BookBuddy Rede Social
**BookBuddy** é uma rede social simples focada em compartilhar opiniões sobre livros. Conecte-se com novos amigos, deixe suas opiniões e organize seus livros em listas.

## Funcionalidades
- Citações de livros através da **API Recite**;
- Seguir e deixar de seguir usuários;
- Manter curtidas nas postagens;
- Buscar livros através da **API Google Books**;
- Manutenção da lista;
- Perfil do usuário, com número de seguidores, seguindo e livros salvos na lista;

## Pré-requisitos
- **Node.js**: verifique a instalação no [site oficial](https://nodejs.org/en/download)
- **Expo Cli**: verifique a instalação no [site oficial](https://docs.expo.dev/workflow/expo-cli/)

## Instalação
- **Baixar dependências**
```bash
npm install
```

- **Executar o projeto localmente**
```bash
npm start
```

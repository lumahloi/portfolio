# MovieNews
Site que busca filmes e retorna notícias relacionadas utilizando as APIs **OMDBApi** e **NewsApi**.

## Funcionalidades
- Busca e exibição de filmes por título;
- Exibição de detalhes e notícias do filme selecionado;
- Exibição do histórico de buscas;
- Favoritar filmes.

## Pré-requisitos
- **Java JDK 8**: verifique a instalação no [site oficial](https://maven.apache.org/install.html)
- **Apache Maven 3.6+**: verifique a instalação no [site oficial](https://www.oracle.com/br/java/technologies/javase/javase8-archive-downloads.html)

## Instalação
- **Compile o projeto Maven**
```bash
mvn clean package
```

- **Execute o servidor Jetty**
```bash
mvn jetty:run
```

# Meowverse E-Commerce
**Meowverse** é uma loja virtual que vende estatuetas, guiando o cliente por toda a experiência e finalizando a compra com sucesso. Este projeto foi meu Trabalho de Conclusão de Curso (TCC) para o curso de Técnico em Desenvolvimento Web que fiz, sendo, portanto, meu primeiro projeto fullstack. A modularização e o design do projeto não são os melhores e serão corrigidos no futuro, assim como a lógica do código e a refatoração.

## Funcionalidades
- Busca de produtos: por nome, categoria, preço, disponibilidade, série;
- Cadastro, login e logout;
- Histórico de compras;
- Remoção e adição de carrinho, uso de cupons;
- Detalhes do produto;
- Pagamento via Pix, cartões de débito/crédito e boleto.

## Como executar localmente
Guia passo a passo de como executar o projeto localmente em sua máquina.

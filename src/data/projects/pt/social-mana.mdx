# Mana Rede Social

**Mana** é a sua nova rede social, criada para conectar pessoas de maneiras autênticas e significativas. Aqui, você pode compartilhar suas histórias e se inspirar em conteúdos que realmente importam. Faça parte de uma comunidade vibrante e acolhedora!

## Funcionalidades
- Login, logout e login do usuário;
- Criação e exibição de postagens;
- Fazer e desfazer votos positivos/negativos;
- Criptografia de senha.

## Como executar localmente
Guia passo a passo de como executar o aplicativo localmente.

### Pré-requisitos
- **Node.js**: verifique a instalação no [site oficial](https://nodejs.org/en/download)
- **MySQL**: verifique a instalação no [site oficial](https://dev.mysql.com/downloads/installer/)

### Instalação
- Acesse o diretório ````frontend```
```bash
cd frontend
```
- Baixe as dependências do frontend
```bash
npm install
```

- Acesse o diretório ````backend```
```bash
cd ../backend
```
- Baixe as dependências do backend
```bash
npm install
```

### Executando o projeto localmente
- Acesse o diretório ````frontend```
```bash
cd ../frontend
```

- Em um terminal, Execute o frontend:
```bash
npm run dev
```

- Em outro terminal, acesse o diretório ```backend```
```bash
cd ../backend
```

- Em um terminal, execute o backend:
```bash
npm run start
```

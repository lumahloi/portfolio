# HtmlAnalyzer

**HtmlAnalyzer** é uma classe Java que analisa o conteúdo da página HTML a partir de uma determinada URL. Ele lê todo o conteúdo e extrai a tag mais profunda.


## Pré-requisitos
- **Java JDK 8+**: verifique a instalação no [site oficial](https://www.oracle.com/br/java/technologies/javase/javase8-archive-downloads.html)

## Compilar
- **Compilar o programa**
```bash
javac HtmlAnalyzer.java
```

## Executar
- **Executar o projeto localmente**
```bash
java HtmlAnalyzer https://example.com
```

My portfolio website used to showcase my most relevant projects. 

## Pre-requisites
- **Node.js**: check installation at the [official website](https://nodejs.org/en/download)

## Installation
- **Download dependencies**
```bash
npm install
```

- **Run  the project locally**
```bash
npm run start
```
